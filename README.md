Voice Rotor PC UDP

Software para la lectura por voz de la posición del rotor desde PstRotator.

REQUISITOS
- Windows
- PstRotator instalado y funcionando

CONFIGURACIÓN DE PSTROTATOR
1. Abrir PstRotator
2. Aceptar la apertura del Firewall
3. Menú: Communication → UDP Control Port
4. Configurar el puerto: 12000
5. Configurar Remote IP: 127.0.0.1
6. Abrir Setup
7. Activar UDP Control

USO
1. Ejecutar VoiceRotorPC_UDP.exe
2. Configurar:
   Host: 127.0.0.1
   Puerto: 12000
3. Pulsar START

El programa anunciará la posición del rotor.

NOTAS
- No interfiere con el funcionamiento normal de PstRotator
- No requiere puertos serie ni software adicional
- Soporta idiomas IT / EN / ES
