Voice Rotor PC UDP

Software per la lettura vocale della posizione del rotore da PstRotator.

REQUISITI
- Windows
- PstRotator installato e funzionante

CONFIGURAZIONE PSTROTATOR
1. Aprire PstRotator
2. Menu: Communication → UDP Control Port
3. Impostare porta: 12000
4. Impostare Remote IP: 127.0.0.1
5. Abilitare UDP Control

UTILIZZO
1. Avviare VoiceRotorPC_UDP.exe
2. Accettare l'apetura del Firewall
3. Impostare:
   Host: 127.0.0.1
   Porta: 12000
4. Premere START

Il programma annuncerà la posizione del rotore.

NOTE
- Non interferisce con il funzionamento di PstRotator
- Non richiede porte seriali o software aggiuntivi
- Supporta IT / EN / ES
